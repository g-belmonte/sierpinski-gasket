#+title: Sierpinski gasket
* Installing
  First make sure to have OpenGL, glew, glfw and glm installed on the system. E.g.: on Archlinux:

  #+BEGIN_SRC shell
  $ sudo pacman -S glew-wayland glfw-wayland glm
  #+END_SRC

  Then:

  #+BEGIN_SRC shell
  $ make && ./sierpinski
  #+END_SRC
